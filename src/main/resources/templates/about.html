Características del DataWarehouse
  • Evaluación de riesgo económico
  • Mapas de riesgo sísmico zonificado
  • Análisis de distribución de magnitudes
  • Monitoreo en tiempo real
  • Reportes y estadísticas avanzadas

Arquitectura del Sistema

Capa de Datos

MySQL con tablas optimizadas para sismos, dimensiones poblacionales, económicas y temporales.

Capa de Negocio

Spring Boot con servicios especializados para análisis sísmico y gestión del DataWarehouse.

Capa de Presentación

Mapas interactivos con Leaflet, gráficos con Chart.js y interfaz responsiva.

Fuentes de Datos

Datos Sísmicos
  • Servicio Sismológico Nacional (SSN) - UNAM
  • Red Sismográfica Nacional
  • Catálogos históricos de sismos
  • Datos de magnitud, ubicación y profundidad
Datos Socioeconómicos
  • INEGI - Censo de Población y Vivienda
  • INEGI - Datos económicos por estado
  • Información de PIB y sectores productivos
  • Densidad poblacional y demografía
